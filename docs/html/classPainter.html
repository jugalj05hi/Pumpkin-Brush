The PainterThreadFunction represents the function the painterThread will execute. It will allow a painterThread to enqueue command information into a queue and then notify the picassoThread that it needs to send that command to the relevant clients.

Parameters
socketPa unique-pointer representing a TcpSocket from the SFML library, this is the socket the thread will use to receive information, it will be stored in a map with the socketIndex as the key.
socketIndexthe key within the map that will have the socketP as the value.
